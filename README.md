## 📈 Methodology

The dashboard uses interpretable macro signals including:

- Macro liquidity conditions
- Real-yield changes
- USD momentum
- Gold timing signals
- Crypto momentum (capped satellite)
- Regime detection & robustness checks

---

## ✅ Validation Framework

| Validation Method | Purpose |
|-------------------|---------|
| Same-period benchmark comparison | Relative performance assessment |
| Transaction-cost drag | Realistic return simulation |
| Non-overlapping 3M returns | Reduced overfitting |
| Train / validation / test split | Out-of-sample testing |
| Walk-forward parameter validation | Adaptive robustness |
| Sensitivity analysis | Parameter stability check |
| Regime breakdown | Performance across market environments |
| Signal contribution | Attribution analysis |
